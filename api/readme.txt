Pada aplikasi berisi:
1. form input --> untuk memasukkan judul buku
2. tabel daftar klasifikasi --> tabel ini menampilkan nomor klasifikasi mulai 000-900, jenis klasifikasinya dan keterangan dari tiap nomor klasifikasi
tabel ini berfungsi untuk membantu mengetahui apakah hasil klasifikasi sesuai dengan kaidah pengklasifikasian buku

Cara penggunaan :
1. inputkan judul buku pada form input
contoh judul buku : Budi Pekerti Ketika Makan
2. Klik button klasifikasikan berwarna pink
3. Hasil klasifikasi berwarna ungu akan muncul diatas tabel daftar klasifikasi
contoh hasil : buku berjudul "Budi Pekerti Ketika Makan" termasuk klasifikasi nomor 100

dapat dilihat pada tabel, bahwa nomor 100 merupakan jenis klasifikasi filsafat dan psikologi, dimana pada keterangan buku yang membahas tentang etika/kepribadian seperti buku "Budi Pekerti Ketika Makan", benar masuk kedalam nomor klasifikasi 100
